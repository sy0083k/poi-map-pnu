# 소개 영상 샷리스트 (지자체/공공 실무자 대상)

프로젝트: 관심 필지 지도 (POI Map PNU)  
작성일: 2026-03-09  
권장 길이: 60-90초 (기준안 75초)

## 목적
- 시청 10초 이내에 서비스 가치(탐색·업로드·현장사진 연계)를 전달한다.
- `/siyu`, `/file2map`, `/photo2map`, `/admin` 핵심 흐름을 한 번씩 보여준다.
- 기술 과시보다 실무 효율 메시지를 우선한다.

## 제작 사양
- 화면비: 16:9 (기본), 필요 시 9:16 재편집
- 영상 템포: 컷당 2-6초, 기능 시연 컷 6-12초
- 자막 규칙: 한 줄 15자 내외, 핵심 동사 중심
- 보안 규칙: 계정/비밀번호/토큰/개인정보/로컬 경로 노출 금지

## 샷리스트 (75초)
| 구간 | 화면(조작) | 자막 | 내레이션 |
|---|---|---|---|
| 0-6초 | `/siyu` 첫 화면(지도+사이드바+상단 메뉴) 전체 보여주기 | 공유재산 관리, 파일과 지도 따로 보시나요? | 관심 필지 지도로, 현황 파악부터 탐색까지 한 번에 진행합니다. |
| 6-20초 | 주소/면적/재산용도/지목/재산관리관 필터 입력 → 목록 선택 → 상세 패널 표시 | 복합 필터 검색 + 선택 필지 상세 확인 | 조건을 한 번에 검색하고, 선택한 필지의 상세 정보를 즉시 확인합니다. |
| 20-36초 | 상단 메뉴로 `/file2map` 이동 → 엑셀 업로드 → 하이라이트 반영 확인 | 엑셀 업로드 후 즉시 지도 반영 | 보유한 엑셀 목록을 업로드하면 관심 필지가 지도와 목록에 바로 연결됩니다. |
| 36-50초 | `/photo2map` 이동 → 폴더 선택 → GPS 사진 마커 생성 → 목록 선택/미리보기 | 현장 사진의 GPS를 지도에서 확인 | 현장 사진 위치를 자동 인식해, 사진과 필지를 함께 검토할 수 있습니다. |
| 50-64초 | `/admin`에서 시유지 업로드/연속지적도 업로드 UI 짧게 시연 | 관리자 업로드로 데이터 최신화 | 관리자 화면에서 데이터와 지적도를 업데이트해 최신 상태를 유지합니다. |
| 64-75초 | 메인 지도 복귀, `/siyu` `/file2map` `/photo2map` 메뉴 강조 후 종료 | 탐색·업로드·현장확인을 하나의 워크플로로 | 관심 필지 지도, 공공 자산 관리 업무를 더 빠르고 일관되게 만듭니다. |

## 편집 메모
- 배경음은 낮은 볼륨으로 유지하고 내레이션 가독성을 우선한다.
- 전환 효과는 짧은 컷(150-250ms) 위주로 사용한다.
- 기능 설명이 겹치지 않도록 자막과 내레이션 문구를 동일 반복하지 않는다.

## 검수 체크리스트
- 10초 이내에 서비스 목적이 이해되는가?
- 4개 흐름(`/siyu`, `/file2map`, `/photo2map`, `/admin`)이 모두 노출되는가?
- 실제 동작 순서(업로드→반영, 선택→상세)가 제품과 일치하는가?
- 민감정보 노출이 없는가?
- 75초 기준 메시지 과밀 없이 이해 가능한가?
